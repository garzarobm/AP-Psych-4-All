INSTRUCTIONS
===============================
-- Insert the CD-ROM into your computer's disk drive.
-- On a Macintosh, The CD-ROM will display its contents. Double click the "AP Psychology.dmg" and then click on the setup icon to install the application. In case of updating/re-installing the application again, it is recommended to remove the existing application before installing a new version.
-- On a PC, the CD-ROM should auto-launch. If it does not, go to the "My Computer" window and double-click the program icon; then double-click the AP Psychology.exe icon to launch.

Note: This application will bookmark your progress and save your test results to your computer's hard drive.
